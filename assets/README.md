# ox-jsdoc Logo Assets

This directory contains logo assets for ox-jsdoc.

## Files

| File                | Description              | Use Case                              |
| ------------------- | ------------------------ | ------------------------------------- |
| `logo.svg`          | Primary square icon mark | Favicons, app icons, compact surfaces |
| `oxjsdoc-light.svg` | Light wordmark lockup    | Dark-mode README, docs, and badges    |
| `oxjsdoc-dark.svg`  | Dark wordmark lockup     | Light-mode README, docs, and badges   |

## Design Concept

The ox-jsdoc logo combines three key concepts:

1. **Document surface** - A single document shape represents JSDoc source comments and documentation metadata.
2. **Comment syntax** - The `/**`, `@`, and `*/` symbols connect the mark directly to JSDoc block comments and tags.
3. **Oxc-style lockup** - The wordmark uses a bold technical lockup with the symbol presented as a compact accent.

## Color Palette

| Color       | Hex       | Usage                                  |
| ----------- | --------- | -------------------------------------- |
| Cyan        | `#0891b2` | Primary icon document surface          |
| Bright Cyan | `#32f3ef` | Accent mark and Oxc-style highlight    |
| White       | `#ffffff` | Light glyphs and dark-surface wordmark |
| Black       | `#000000` | Light-surface wordmark                 |

## Usage Guidelines

- Maintain aspect ratio when scaling.
- Use `logo.svg` for square or compact placements.
- Use `oxjsdoc-light.svg` on dark backgrounds.
- Use `oxjsdoc-dark.svg` on light backgrounds.
- Ensure the icon symbols remain legible at small sizes.
